# Auth Arcane Gaming

## Установка зависимостей
```sh
npm install
```

### Установка переменных среды окружения перед запуском проекта
>Создайте файл `.env` в корне проекта на основе файла `.env.example`.

### Запуск в режиме разработки
```sh
npm run dev
```

### Сборка проекта
```sh
npm run build
```

### Lint
```sh
npm run lint
```
